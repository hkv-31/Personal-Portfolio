# Portfolio Website

A minimal, fully responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Responsive Design:** Optimized for mobile, tablet, and desktop.
- **Smooth Animations:** Interactive elements and smooth scrolling.
- **Modern Aesthetic:** Clean and professional UI.

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
